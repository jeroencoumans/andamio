Helper to generate striped list. The first item will not get a top border and the last item will not get a bottom border.

.striped-list(@top-border, @bottom-border)

Sets the list items to use the specified top and bottom borders. Example: see .media-object-list

  • @top-border: sets the color of the top border (default: #fff)
  • @bottom-border: sets the color of the bottom border (default: @grey)
