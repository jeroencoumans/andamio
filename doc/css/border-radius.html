.border-radius(@radius: 2px)

This mixin allows you to set the default for rounded corners.

  • .border-radius(@radius): specify a unit (e.g. 5px or @grid-s) to set all 4 corners to the same rounded border
  • .border-radius(@radius @radius @radius @radius): allows you to set different units for every corner (top left, top right, bottom right, bottom left)
