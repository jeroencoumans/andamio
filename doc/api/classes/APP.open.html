page ( url view [refresh] [callback] )

Defined in js/app/open.js:47

Do an AJAX request and insert it into a view. This method also maintains the URL's for each view, and sets the activeUrl to the called URL.

Parameters:

url String

the URL to call

view HTMLElement

what page to insert the content int (child, parent or modal)

[refresh] Boolean optional

when set, the activeUrl will be downloaded again. You need to set this parameter if you want to explicitly refresh a page.

[callback] Function optional

optional callback function that will be called when the AJAX call completes
