getQueryParam ( name defaultValue queryString )

Defined in js/app/util.js:9

Returns the value for a given query string key.

Parameters:

name String

Query string key

defaultValue String

If the query string is not found it returns this value.

queryString String

Query string to pick the value from, if none is provided window.location.search query string will be used. This parameter makes the function testable.

Returns:

The value of the query string or defaultValue if the key is not found. If the value is empty an empty string is returned.
