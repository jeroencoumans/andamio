On mobile, the page navigation is hidden by default. You can add a toggle inside the .page-header to let the user show or hide it.

The following class will let you create an element that will toggle the menu:

.action-show-nav

Example that can be used in a .page-header:

Menu

Structure of the page navigation:

.page-navigation
.navigation-header
Used to separate any navigation blocks with a header. Preferably, use a h2 heading for this.
.navigation-item
Link in the navigation.

The .navigation-item can have the following (optional) classes:

.action-nav-item
Adding this class will let Andamio lookup the link and insert it into the parent view. Additionally, Andamio will make sure the selection navigation item is highlighted.
.active
Highlight for the selected (active) navigation item.
