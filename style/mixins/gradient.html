.gradient(@startColor : @white, @endColor: @grey, @startPoint: 0, @endPoint: 100%)

Allows you to easily create a vertical gradient background.

  • @startColor: the top color (required)
  • @endColor: the end color (required)
  • @startPoint: where the gradient should start (0-100%, default 0)
  • @endPoint: where the gradient should end (0-100%, default 100%)
