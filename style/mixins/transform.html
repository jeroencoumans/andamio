.translate3d(@x, @y: 0, @z: 0)

Easily reposition an element on the x-axis, y-axis and z-axis. HACK: specify this without arguments to give an element hardware-acceleration on iOS.

  • @x: transform an element on the x-axis
  • @y: transform an element on the y-axis
  • @z: transform an element on the z-axis
.scale(@ratio)

Scale an element by multiplying it by the @ratio.

E.g. shrink it 50%: .scale(0.5)

E.g. grow it 20%: .scale(1.2)

.rotate(@deg)
Rotate an element by 0-360 @deg
