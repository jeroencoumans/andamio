This describes the structure of the HTML document.

The html element can contain one of the following classes:

  • .website: for mobile website
  • .webapp: for mobile web application
  • .desktop: for desktop website
  • .desktop.webapp: for desktop web application

The .viewport class is mandatory for the body element.

After that, we use the following DOM structure:

.page-view
  • .parent-view
  • .child-view
The page-view is a wrapper for the parent-view and any (optional) child-view.
.page-navigation
Contains navigation elements. On mobile, it will appear as a slide-out menu from the left. On desktop, it will remain fixed to the left.
.page-navigation-toggle (mobile-only)
This link overlaps the content when the navigation is open. This makes it easier to close the menu just by tapping anywhere outside it.
.modal-view
Wrapper for any modals. These will slide up from the bottom.
.loader
When any content takes longer than half a second to load, a spinner will appear on top of the content. The loader contains the spinner.
