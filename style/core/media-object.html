These classes will produce styles that allow you to create various types of content that feature a left-aligned image, some content and optional actions on the right.

It's a very simple grid system with flexible dimensions and suitable for things like newsfeeds, comments, etc.

.media-object
Wrapper for each media object. Can be a div or link (if you want to make the media-object clickable).
.media-img
Optional. Image (really, any kind of media, so icons, thumbnails etc.) that appears on the left side.
.media-action
Optional action (e.g. a link, button, dropdown, etc.) that will be displayed on the right. If you want text to wrap around this, put it inside the .media-bd.
.media-bd
Content for the media object
