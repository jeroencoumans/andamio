Header buttons

We also have header buttons. These appear inside the page-header with a wrapper around it. We do this to increase the click target of the button.

There are two wrappers: button-header-wrapper (left-aligned header button) and button-header-wrapper-alternate (right-aligned header button).

Regular Alternate

You can add the button-primary class here as well. Make sure to do it on the actual button and not on the button-header-wrapper
